def geocode(address,city=None,batch=None,sig=None)->dict:
    """获取地理编码"""
    url = 'https://restapi.amap.com/v3/geocode/geo?parameters'
    params={
        'key': key_chen,
        'address':address,
        'city':city,
        'batch':batch,
        'sig':sig,
        'output':'json'
    }
    response = requests.get(url,params=params)
    data = response.json()
    return data

总结¶

思考¶

  • AI:人工智能
  • ML:Machine Learning 机器学习

在调用高德中每一个功能的同时，其背后大多含有ML、AI的功能，她们隐藏在每一个细小的环节背后。

  • 高德多个模块通过对地址进行分析，提取字符信息返回相对应的信息。

  • 通过AI算法对路网、路段、路口的交通运行情况进行分析。在交通态势之中，只需填写道路名称便能获取道路的拥堵情况、可行驶速率等相关的条件进行的预测评估等详细的说明。

  • 在路径规划、搜索POI等功能模块通过对图像/视频的分析、诊断、搜索、合成等方面的算法研究进行车辆和人员的检测识别与跟踪
  • 在计算机视觉方面，高德运用图像分割的基础技术，在各个数据自动化生产线中得到了广泛应用，助力高德地图的高度自动化数据生产。在静态地图等功能中都能有所体现。
